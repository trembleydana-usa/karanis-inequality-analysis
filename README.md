# karanis-inequality-analysis
Udacity project 1: Analysis of inequality in Roman Egypt using Karanis tax records

# Inequality in Ancient Egypt: Evidence from a Roman Village

## Overview
This project analyzes tax payment data from Roman Egypt (Karanis) to understand economic inequality.

Using over 17,000 transactions, tax payments are used as a proxy for economic activity to answer:

- How are payments distributed?
- How unequal is the distribution?
- Who contributes most?
- Can differences be explained?

---

## Dataset

Source: Karanis Tax Rolls Database (Oxford Roman Economy Project)

- 17,000+ transactions
- 1,000+ individuals
- Includes tax type, payment value, and date

## Summary of Results

- Tax payments are highly unequal (Gini ≈ 0.58)
- Top 10% contribute ~34% of total payments
- Bottom 50% contribute ~8%
- Model explains only part of variation
